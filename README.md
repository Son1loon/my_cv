## ✨ Features

- 🩸 **Unique Blood Effect** - Dynamic particles and blood drops that react to mouse movement
- 📱 **Fully Responsive** - Perfect layout on all devices (desktop, tablet, mobile)
- 🎯 **Mobile Menu** - Convenient hamburger navigation on phones
- ⌨️ **Typing Animation** - Dynamic role switching (Developer, Programmer, Maker, Moderator)
- 🎨 **Smooth Animations** - Elements fade in as you scroll
- 🌈 **Gradient Design** - Stylish blood-themed gradients
- ⚡ **Performance Optimized** - Smooth 60fps animations
- 🎯 **Active Menu Highlight** - Current section is highlighted in navigation
- 🔗 **Social Links** - Easy access to GitHub, Email, and Phone
